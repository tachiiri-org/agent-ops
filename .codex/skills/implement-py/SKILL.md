# implement-py skill

## Goals

- Plan changes for goals
- Implement changes
- Format and lint by `ruff format` and `ruff check`
- Typecheck by `pyright`
- Unit test by `pytest`
- After pass all, commit changes

## Constraints

- do not create pr
